How to install
Up and reading in 5 minutes

No account needed. No subscription. Just a free Gemini API key and a manga site.

1
Download the extension zip from GitHub Releases and unzip it
2
Go to chrome://extensions → Enable Developer Mode → Load Unpacked
3
Get a free Gemini API key at aistudio.google.com (takes 30 seconds)
4
Click the extension icon, paste your key, pick your language → Start reading

Privacy Policy

Last updated: May 2025

What MangaLens does

MangaLens is a Chrome browser extension that translates text inside manga and manhwa images using the Google Gemini Vision API.

Data we collect

MangaLens does not collect, store, or transmit any personal data. Specifically:

  • Your Gemini API key is stored locally in Chrome's storage and is never sent to any server other than Google's API endpoint.
  • Images from manga pages are sent directly to the Google Gemini API for processing. They are not stored or logged by MangaLens.
  • Translation memory (character names and terms) is stored locally in your browser's storage only.

Third-party services

MangaLens uses the Google Gemini API to process images. When you use MangaLens, image data is sent to Google's servers under Google's own privacy policy. You can review Google's privacy policy at policies.google.com/privacy.

Permissions

MangaLens requests the following Chrome permissions:

  • storage — to save your API key and language preference locally
  • activeTab — to scan images on the page you are reading
  • scripting — to inject the translation overlay into manga pages
  • host_permissions (<all_urls>) — to fetch manga images through the service worker, which bypasses cross-origin restrictions that would otherwise prevent reading image data

Contact

If you have questions, open an issue at github.com/Ayush-1271/MangaLens.
